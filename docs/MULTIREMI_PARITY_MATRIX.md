# Multiremi Bun Parity Matrix

This matrix tracks whether the native Bun/TypeScript Multiremi implementation can replace the current Go Multica server behavior. `done` requires contract coverage, behavior coverage, tests, and a local smoke path.

## Legend

- `done`: implemented, tested, and smoke-verified.
- `partial`: code exists, but contract, auth, edge cases, or smoke verification are incomplete.
- `missing`: no usable implementation in `src/multiremi`.
- `reference`: source material only; not a runtime dependency target.
