选择你偏好的映射关系展示方式

A

流向图 (Flow Diagram)

左右两列：Source 在左，Target 在右，用连线表示映射关系。按分组（Projects / Globals / Wiki）折叠展示。一眼看清文件系统布局。

B

增强表格 (Enhanced Table)

恢复原有表格，但增加：分组标签、路径缩写 hover 全路径、状态颜色编码、inline 修复按钮。实用为主。

GROUPSOURCETARGETSTATUS
Global ~/.claude/CLAUDE.md → ~/.remi/soul.md ● OK
Global ~/.claude/skills/ → ~/.remi/skills/ ● OK
Project ~/.claude/projects/-data00-…/ → ~/.remi/projects/-data00-…/ ● Broken
C

混合模式 (Flow + Table)

顶部用流向图展示分组概览（Globals / Projects / Memory），点击某个分组展开为详细表格。兼顾全貌和细节。
