A. Tree Sidebar + Content Panel

Left sidebar shows memory tree (Personal / Projects), right panel shows filtered entities. Click tree node to filter, click entity to open full page.

Pros

  • Clear hierarchy at a glance
  • Fast navigation between scopes
  • Familiar file-explorer pattern

Cons

  • Takes horizontal space
  • May be overkill if few projects

B. Scope Tabs + Type Groups

Top tabs switch between All / Personal / per-Project scope. Entities grouped by type within each scope. Click entity to open full page.

Pros

  • Simple, flat navigation
  • Easy to add new projects as tabs
  • No sidebar space cost

Cons

  • Many projects = too many tabs
  • No visual hierarchy

C. Chip Selector + Full-Page Detail (Recommended)

Scope chips at top (Personal / each Project / MEMORY.md / Daily). Click chip to filter scope. Click entity navigates to full-page detail view. Keeps current list UX but adds project awareness.

Pros

  • Minimal layout change from current
  • Clear scope switching via chips
  • Full-page detail (no more sheet)
  • Scalable: chips scroll horizontally

Cons

  • Less visual hierarchy than tree
